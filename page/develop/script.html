# 1、安装依赖

可以使用 npm、cnpm、yarn 等进行安装。

npm install
1

# 2、编译、示例

# 2.1 编译 SDK

npm run build
1

# 2.2 编译 SDK 和文档

npm run build:all
1

# 2.3 使用示例

npm run example
1

此命令将在本地使用 http-server 启动两个 Http 服务分别监听 18080 端口和 28080 端口。18080 端口监听 examples 目录，28080 端口监听 lib 目录。开发过程中可以在 examples 目录下编写对应的 html 文件然后访问 18080 端口来测试 SDK 的功能是否正常。

# 3、文档

基于 Vuepress 搭建项目的文档。

# 3.1 编写文档

文档使用 MarkDown 编写，这种方式可以边修改 MarkDown 文件边自动编译。可以即时看到最新的文档。

npm run docs:dev
1

# 3.2 编译文档

用来将 MarkDown 源文档编译成 HTML 等静态资源文件，用于文档编写完成后的部署使用。编译后的文档可以部署到 nginx 等 HTTP 服务中。

npm run docs:build
1

进入 docs/.vuepress/dist 目录查看项目详细文档。
