# 项目简介

本项目用于移动端H5页面、小程序页面、WebView和PC端浏览器页面运营数据的采集、采集信息包括PV、UV、用户设备信息、页面停留时长、用户访问路径等。

整个项目的前端由采集探针（trigger项目）和数据管理后台两部分组成，目前数据管理后台内嵌在运营后台、Boss后台等管理后台中、没有单独的后台项目。当用户进入浏览器页面后、引入了采集探针的项目将会自动采集当前浏览器的设备信息及该用户的PV、UV等数据，并在适当的时候通过接口将采集到的数据上传。采集探针上传的数据可以在对应的管理后台中查看。

采集探针即 trigger 项目，使用 TypeScript 作为开发语音、 Webpack 作为构建工具。
